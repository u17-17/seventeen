# YAN TUTOR

极简黑白灰风格的个人教师品牌前端页面，面向高一、高二学生和家长，展示高中数学 / 高中物理一对一辅导服务。

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Development

```bash
npm install
npm run dev
```

默认开发地址：

```text
http://localhost:5173/
```

## Build

```bash
npm run build
```

## WeChat QR

将真实二维码图片放入：

```text
public/wechat-qr.png
```

页面会自动替换预约区的二维码占位框。
