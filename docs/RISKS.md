# 风险登记册

等级：P0 阻塞目标，P1 主要风险，P2 可管理风险，P3 观察项。  
任何风险接受、降级或关闭都需要证据，并在 `DECISIONS.md`/`CHANGELOG.md` 留痕。

## 风险处理优先级

1. 在任何开发前关闭 R-004，并确认 R-003/R-005/R-006 的事实和决策门；
2. 官网开发首先处理 R-001/R-002/R-008/R-009；
3. 外部发布前处理 R-005/R-010/R-015/R-016；
4. 豆包测试前处理 R-011/R-012/R-013/R-020/R-021/R-022/R-024；
5. 四层方法公开页面或地图渠道进入范围前处理 R-023/R-026；
6. 首次闭环后再评估 R-018/R-019/R-025 的更大优化与自动化。
