标题：

涉县高一高二数学物理一对一：第一次学习诊断要看什么

摘要：

面向高一、高二学生家长，说明在安排高中数学、物理一对一辅导前，第一次学习诊断应该准备哪些信息、观察哪些问题，以及涉县线下和邯郸线上服务边界。

正文：

很多家长找高中一对一辅导时，会直接问“能不能补数学”“物理能不能提上来”。但真正开始排课前，我更建议先做一次学习问题诊断。

原因很简单：同样是分数不稳定，背后的卡点可能完全不同。有的学生是基础知识不牢，有的是题型识别慢，有的是会听课但不会独立写第一步，还有的是考试后没有把错因复盘清楚。如果没有先判断问题在哪里，后面很容易变成重复刷题。

第一次沟通时，建议家长先准备这几类信息。

第一，学生年级和咨询科目。

目前我正式面向高一、高二学生，主要做高中数学、高中物理的一对一学习诊断与辅导。升高一咨询可以沟通学习方式过渡，但不把它扩展成长期正式服务年级。

第二，最近一段时间最明显的卡点。

数学可以先看：读题时能不能标出条件，函数或大题有没有切入点，做题时第一步是否经常卡住。物理可以先看：受力分析、电学基础、模型题过程是否容易混在一起。

第三，不要只看分数，也要看过程。

有些学生看起来“都会”，但一到独立做题就没有思路。这个时候要观察的是：孩子能不能说清题目目标，能不能把已知条件整理出来，能不能说明为什么选这一种方法。

第四，第一次沟通不需要提交敏感信息。

准备年级、科目、当前成绩区间、主要问题和可沟通时间即可。不要提交学生姓名、学校、班级、详细地址、联系方式截图或其他可识别信息。

第五，先判断是否适合，再沟通课程安排。

我不公开具体课时价格，也不做“保证提分”的承诺。是否适合继续安排课程，需要根据学生实际问题、沟通时间和学习目标再判断。

## 头条号草稿（BYTE-TEXT-v1）

标题：

高一数学听得懂但做不出，高二物理模型题容易乱，第一次诊断先看什么

摘要：

一对一辅导不要一开始就急着排课。先看学生的读题、条件整理、第一步选择、模型识别和错因复盘，再判断适合怎样补。

正文：

有些高一、高二学生的问题很像：上课能听懂，老师讲例题也能跟上，但自己做题时就是没有思路。数学大题不知道第一步怎么写，物理综合题一看到模型就容易乱。

这类情况不一定是“不努力”，也不一定是“基础全都不会”。第一次一对一沟通时，我通常会先看几个更具体的点。

1. 学生能不能读清题目目标

很多学生不是不会公式，而是没有先判断题目要什么。数学题里目标、条件、限制范围要分开看；物理题里对象、过程、受力或电路关系要先拆清楚。

2. 能不能独立写出第一步

如果听讲没问题，但自己写题一直卡在第一步，说明问题可能在题型识别和方法选择上。这个时候只继续听更多讲解，未必能解决根本问题。

3. 错题有没有复盘到原因

错题不是抄一遍答案就结束。要看错误来自基础知识、审题、步骤、模型选择，还是计算和书写习惯。不同原因，对应的训练方式也不同。

4. 数学和物理的诊断重点不同

数学更常见的是条件整理、函数题型、目标反推和大题切入点。物理更常见的是受力分析、电学基础、能量或模型题过程混乱。

5. 家长第一次沟通准备什么

准备学生年级、咨询科目、当前成绩区间、主要卡点和可沟通时间即可。不需要提交学生姓名、学校、班级、详细地址、聊天截图或成绩单原图。
