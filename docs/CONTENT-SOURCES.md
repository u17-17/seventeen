# 内容与信源台账

用途：记录官网、外部平台、收录状态、证据和豆包可见来源。本文当前只登记公开现状，不代表用户已授权外部发布。

## 1. 状态词

- `计划中`：尚未创建；
- `草稿`：内容存在但未发布；
- `已发布`：URL 公开可访问；
- `已发现`：搜索结果或站长工具能发现 URL；
- `已收录`：搜索结果可通过稳定查询找到；
- `已更新`：公开内容已修订；
- `已撤回`：不再公开；
- `待核验`：访问、事实、授权或收录证据不足。
- `豆包可见`：豆包回答的可见来源中出现该 URL；不等于已证明因果贡献。

“公开可访问”“被发现”“被收录”“被豆包显示为来源”“回答主动提及实体”是五个不同状态，不得混用。
